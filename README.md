# StructKinator
A data structure chooser with 8 questions


Structkinator es una aplicación educativa que guía a los usuarios en la selección de la estructura de datos más apropiada para sus necesidades específicas. Basado en un sistema de preguntas y respuestas, proporciona recomendaciones fundamentadas con visualizaciones, pseudocódigo y ejemplos prácticos.

- **Arreglos**: Estáticos y Dinámicos
- **Listas Ligadas**: Simples y Dobles
- **Pilas (Stack)** y **Colas (Queue)**
- **Árbol Binario de Búsqueda (BST)**
- **Heaps** / Colas de Prioridad
- **Grafos** (Lista de Adyacencia)
- **Tries** (Árbol de Prefijos)
